## 推断启发

- 一篇内容**多个角度**时允许多选，最多 3 个 category
- 优先用两级（更具体），找不到合适子类才回退扁平
- AI/编程/技术类 → `科学与技术` 或 `计算机科学`
- 投资/财经/商业 → `经济与投资` / `经济理财-财经` / `经济理财-商业`
- 心理/认知 → `心理-认知与行为` / `个人成长-认知思维` / `心智与方法`

## 新增 category 时的提示模板

```
> [!note] 本次新增 category 候选
> 在已有词表中没找到合适项，建议新增：`xxx-yyy`
> 如确认，把它加到 references/category-pool.md
```
